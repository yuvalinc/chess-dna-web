How posting works (no auto-post)
The extension never submits a comment on its own. Every post takes one click from you on Reddit's own Submit button. Flow:
  1. Open any matching Reddit thread.
  2. An overlay appears bottom-right with the AI draft.
  3. Click Fill comment box → draft fills Reddit's reply textarea.
  4. You review / edit / click Reddit's own Comment button.
  5. Click Mark posted in the overlay so the dashboard ratio gate updates.
Karma sync and reply detection are read-only — no posting either way.
