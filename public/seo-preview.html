Checklist itemWhat scanner readsHow to fix

Description is too vague (16 words) Base44 app-description setting Edit app description in Base44 Settings, ≥20 words / ≥80 chars
App has no content for AI Urgent Base44 Page-entity registry Use Base44 "Fix with AI" or create Pages in their dashboard
App has no pages Urgent Same Same
No pages to list in the sitemap Same Page registry — ignores my shipped sitemap.xml Same
App is missing trust pages Same — only Base44 Pages count, not my about.html Same
