# Listing-inclusion outreach drafts

Drafts of cold-outreach emails to chess editorial sites whose "Best Chess Apps" listicles are cited by Google AI Mode / Gemini / ChatGPT but don't mention Chess DNA. Each draft is ready to copy-paste — Claude can't send email or fill out logged-in forms on the user's behalf, so the actual send is manual.

## How to use

1. Open the draft for the target site.
2. Read the "Channel" note — some sites accept email, others require a logged-in form.
3. Confirm the recipient address by visiting the site footer / "About" / "Contact" page if it's listed as `[needs confirmation]`.
4. Copy the Subject + Body. Send from `yuval@chessdna.app`.
5. Provision a reviewer trial account (preloaded with sample games) before sending — reply-back time is much higher when you offer the demo up-front.
6. Attach 2–3 screenshots: skill radar, pattern breakdown, game review. Drop them in `docs/press/outreach/assets/` if you want to keep them versioned.
7. Tick the after-send checklist at the bottom of the draft.

## Tone guidelines (learned from past pitches)

- Compliment the article without grovelling. Show you actually read it.
- Lead with the *category* you'd slot into, not the product features.
- Differentiator first ("pattern analysis across full history"), feature list never.
- Offer a demo account loaded with sample games — removes the "I don't want to import my games" friction.
- Don't bcc. Don't send the same week to multiple competing sites — risk of one publishing before the other. Stagger by ~3 business days.
